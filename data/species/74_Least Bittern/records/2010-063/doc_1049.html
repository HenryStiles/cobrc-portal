Number:  1

Plumage:  Other/Unknown

Description of the Bird

We never saw any birds.
Behaviors: It was singing and calling from cattails.
Call: It gave a cu, cu, cu, cu, cu song while getting faster towards the end of the song. It also gave a different call. It was more like co-oooo, it was almost dove like (recordings of both, but only one in WAV file so far).
Plumage: unknown

Similar Species Discussion

Black-billed Cuckoo – This song was coming from the marsh, which helps eliminate BBCU. It was lower pitched than BBCU should be and it was not as whistle like as BBCU.
